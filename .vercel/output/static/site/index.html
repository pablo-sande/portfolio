About this site

This site is a work in progress. Let me start by telling you that yes, it is over-engineered. I did it for the purpose of showing my understanding of some technologies. And well... for fun too :P

I could have gone for a full static site and just host it in Vercel like everybody else does. But since this is not a professional app and I don't care about min-maxing, I took a different approach.

This site consists of an Hybrid Astro app running on NodeJS. Being in Hybrid mode means that some pages are Static Site Generated while others are Server Side Rendered. The project is Dockerized and deployed in a humble single node Kubernetes cluster hosted in DigitalOcean.
